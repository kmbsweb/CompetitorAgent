# Pharmaceutical Launch Price Optimizer (US Market)

本プロジェクトは、マルチエージェント・シミュレーション（MAS）を用いて、競合他社の動的な価格反応や市場の処方慣性を考慮した上で、期待収益・利益を最大化する**「最適なローンチ価格（WAC）」**を逆算（最適化）するためのシミュレーターです。

## 1. プロジェクトの概要

製薬企業の製品価値（NPV）を最大化するためには、単なるコスト積み上げや競合との単純比較ではなく、**「価格を上げた際のシェア喪失リスク」と「価格を下げた際の利益率悪化」**のトレードオフを定量化する必要があります。

本ツールは、US市場特有の価格決定メカニズム（PBMによるシェア制御、WACの上昇傾向など）をモデル化し、数百回のモンテカルロ・シミュレーションを通じて、最も頑健な価格戦略を導き出します。

## 2. 数理モデル：価格・シェア相関ロジック

本シミュレーターでは、価格差によるシェア変動を「シグモイド関数（ロジスティック曲線）」を用いてモデル化しています。これにより、極端な「安ければ100%」という仮定を排除し、実務に近い滑らかなシェア移動を再現します。

## 3. パラメーター定義 (Parameter Guide)

シミュレーションを実行する際に設定可能なパラメーターは以下の通りです。

### A. 市場構造パラメーター (Market Structure)

| パラメーター名 | 意味 | 入力例 | 実務上の設定根拠 |
|---|---|---|---|
| comp_prices | 競合他社の現在薬価 | [15000] | 主要なベンチマーク薬の月額WAC価格。 |
| max_share | 到達可能シェア上限 | 0.40 - 0.60 | 既存薬の処方慣性やPBMの制約で「これ以上は取れない」という壁。 |
| min_share | 維持可能シェア下限 | 0.05 - 0.15 | ブランド力や特定の患者層で「高くても維持できる」底堅いシェア。 |
| sensitivity | 価格感受性 (K値) | 4.0 - 10.0 | 数値が高いほど、競合との価格差がシェアに敏感に反映されます。 |

### B. 経済・期間パラメーター (Economics & Duration)

| パラメーター名 | 意味 | 入力例 | 実務上の設定根拠 |
|---|---|---|---|
| cogs | 単位あたりの総原価 | 3000 | 製造コストに加え、PBMへのリベート（Gross-to-Net）を合算した実質コスト。 |
| market_duration | シミュレーション期間 | 24 - 60 | 月次計算の場合。特許失効やIRA交渉までの期間を想定。 |
| iterations | 試行回数 | 200 | 競合のランダムな反応を平均化し、期待値を安定させるための回数。 |

## 4. ユースケース別の設定例

### ケース1：標準的なブランド薬 (Standard Brand)

競合と性能が同等で、適度な価格競争がある場合。

```
max_share: 0.50 / min_share: 0.05 / sensitivity: 8.0
```

### ケース2：高い臨床的価値 (Premium Value)

臨床データが圧倒的で、高くても選ばれる理由がある場合。

```
max_share: 0.80 / min_share: 0.20 / sensitivity: 3.0
```

### ケース3：バイオシミラー/後発品 (Price Driven)

価格の安さだけがスイッチの要因となる場合。

```
max_share: 0.70 / min_share: 0.00 / sensitivity: 12.0
```

## 5. 結果の解釈 (Interpretation)

- **頂点 (Optimal Price)**: 現在のコスト構造と市場環境において、累積利益（NPVの代替指標）を最大化する推奨価格。

- **グラフの傾斜**: 価格を $100 変えた際に、どの程度の利益毀損リスクがあるか（感度）を示します。

- **崖 (Drop-off point)**: 競合に対して価格が高すぎ、市場から急速に排除され始める危険な価格帯を特定します。
